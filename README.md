# KIDZ_POST
Kidz POST has been designed for communication purpose among class teacher and the parents, it is different of existing systems, a teacher can customize his/her parent list, and it is given fully flexible to send messages among parents privately or publicly. Teachers can get the proof whether message is shown or not. And teacher can keep the activities history for future references. This is simple and very helpful for teachers to pass the messages directly to parents and no need to use students as messengers.
